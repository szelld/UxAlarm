## Refactor 01

The goal of this refactor is to apply general design standards to the app and provide a better user experience in general.

It includes both UI and functional changes that affect the current state of the application.
Read the guidelines carefully and plan execution accordingly.

### Colors

This app uses two main colors: BLACK and WHITE.

There shouldn't be any other colors in the app. If a UI component has a default color, rewrite it.

### Background

The background in the app should be WHITE.

On top of the background, there should always be two images:
- sun.png - in the top left corner (with some margin)
- moon.png - in the bottom right corner (with the same margin)


Everything else is placed on top of these.
