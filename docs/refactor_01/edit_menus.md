## Edit Menus

Edit menus serve as both the creation and editing menu.
When the user clicks save, the menu is created (if opened by the add button) or overwritten (if opened by the edit button).

### Background

Switch the side of both the sun and the moon image. Keep the background unchanged otherwise.

### Edit alarm

Alarms should have the following attributes:
- **Time:** The time to wake the user up. It should be set similarly to the current solution but remove the buttons and make it slideable.
- **Sound:** The sound to play (adding the choices is a future project, add some defaults if you can)
- **Repeat:** From Monday to Sunday each button is a toggle.
- **Bedtime reminder:** None or 6 to 10 hours, the current solution is fine.
- **Good Morning Text:** The text to display when the user wakes up.
- **Bedtime Text:** The text to display when we tell the user to wake up.

### Edit reminder

Reminders should have the following attributes:
- **Time:** Same as above
- **Sound:** Same as above
- **Reminder Text:** The text to display when reminding.
